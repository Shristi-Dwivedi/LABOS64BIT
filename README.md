# LABOS - 64-bit Operating System

LABOS is a custom 64-bit operating system written in C and Assembly.

## Features (Current)
- GRUB bootloader
- 64-bit long mode kernel
- VGA text output
- Docker-based build system

## Goal
- GUI system
- Calculator
- Text Editor
- Search engine
- Custom memory manager
- Interrupt system

## Build
make build-x86_64
